Definición sobre las etiquetas en HTML para definir listas y tablas.


Etiquetas para definir listas.


En HTML, hay tres tipos de listas que se pueden definir utilizando diferentes etiquetas. Estas son:


  • Lista ordenada: Se define mediante la etiqueta " < ol > " y cada elemento de la lista se define mediante la etiqueta " < li > " . Esta lista se presenta numerada o con viñetas.

  • Lista desordenada: se define mediante la etiqueta "< ul >" y cada elemento de la lista se define mediante la etiqueta " < li > " . Esta lista se presenta con viñetas.

  • Lista de definición: se define mediante la etiqueta "< dl >" y cada término de la lista se define mediante la etiqueta " < dt > " , mientras que su definición se define mediante la etiqueta " < dd > " . Esta lista se presenta con los términos en negrita y su definición a continuación.

Etiquetas para definir tablas:

Las tablas en HTML se definen utilizando las etiquetas " < table > ,< tr > y < td > " . Estas etiquetas se utilizan de la siguiente manera:

  • " < table > " se utiliza para definir la tabla en sí.

  • " < tr > "se utiliza para definir cada fila de la tabla.

  • " < td > "se utiliza para definir cada celda de la tabla.
